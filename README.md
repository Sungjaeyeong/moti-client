# moti

뭔가를 만들고 싶은데 아이디어가 떠오르지 않나요?

아이디어는 있는데 함께할 팀이 필요한가요?

모티를 통하여 함께할 사람들을 찾아보세요!

moti는 프로젝트를 함께할 팀을 찾을 수 있습니다.
